# estagio-cnpq-projetos
Repositório com projetos desenvolvidos no estágio do CNPq. Inclui automações RPA para extração e processamento de dados em larga escala (Python, Selenium, Pandas) e um sistema full-stack de formulários automatizados (Flask, SQLite, SQLAlchemy, HTML/CSS) para otimização de fluxos internos de dados.
