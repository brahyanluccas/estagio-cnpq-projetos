# Sistema de Gestão de Formulários - CNPq

Aplicação Full-Stack desenvolvida para automatizar a coleta e processamento de formulários internos.

## Tech Stack
- **Backend:** Python com Flask
- **Banco de Dados:** SQLite com SQLAlchemy
- **Frontend:** HTML5, CSS3 e Jinja2
- **Relatórios:** PDF via WeasyPrint

## Funcionalidades
- Cadastro e gestão de usuários.
- Automatização de preenchimento de moldes.
- Limpeza e estruturação de base de dados.
